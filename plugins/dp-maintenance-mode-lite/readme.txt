== Description ==

The DP Maintenance Lite plugin includes the possibility to add a maintenance mode with a sleek theme to your website.

* Elegant & Sleek Design.
* 100% Responsive (Mobile Support)
* Countdown & Progress Bar
* Twitter Feed
* Newsletter Subscription
* MailChimp Support
* Contact Form
* Social Icons
* Set Expiration Date
* Form Validations
* Cross Browser Support.
* Simple to Customize

Get [DP Maintenance PRO](http://wpunderconstruction.com/ "DP Maintenance PRO") for more themes and widgets.

Enjoy !

== Installation ==

1. Copy the "dpMaintenanceLite" folder into your wordpress "wp-content/plugins/" folder using your favorite FTP program.
2. Activate the plugin through the 'Plugins' menu in WordPress

Once the plugin has been activated you will notice a new menu on the side called "Maintenance Lite" enabling you to customize it.
